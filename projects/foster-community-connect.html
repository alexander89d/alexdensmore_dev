Foster Community Connect

This project creates a forum for foster families to indicate needs and potential donors to read about what is needed (and possibly donate those items). I contributed code to this preexisting Open Source project on GitHub as part of my Open Source Software course. See below for more details about and links to each pull request as well as documentation submitted with the course project.
