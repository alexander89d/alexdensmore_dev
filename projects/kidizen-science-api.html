Kidizen Science API

This GitHub repository contains the public API I created as part of a larger project entitled "Kidizen Science." The purpose of the "Kidizen Science" project is to allow students in grades K-8 to participate in Citizen Science projects. The project includes a mobile app for teachers to create projects and students to record observations for those projects as well as this public REST API hosted on Google Cloud Platform with which the mobile app interacts for CRUD functionality in Google Cloud Datastore.
