Church Accessibility Website

This is a website giving advice to churches on being physically and digitally accessible to people with disabilities. The project assignment required creating a simple website and deploying it publicly. While I created this website to fulfill a course requirement, I enjoyed creating a website on something about which I am passionate. This website was originally hosted on Oregon State University's servers. Now that I have graduated, I have posted it within a subfolder on this site (see link in Quick Facts section below).
