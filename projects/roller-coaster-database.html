Roller Coaster Database

This project implements a relational database, server, and frontend website. Through the frontend website, users could perform CRUD (Create, Read, Update, and Delete) operations for various amusement park-related entities: roller coasters, amusement parks, roller coaster features, manufacturers, and park owners. The site was hosted on the Oregon State University servers, and our database accounts were deleted once the course was complete, so the website is no longer live.
